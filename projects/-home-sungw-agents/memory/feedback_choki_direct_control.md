초키봇의 교육 관련 파일 변경은 teacher가 직접 한다. 햄토리를 경유하지 않는다.

**Why:** 마스터가 "초키봇 컨트롤은 너가 해"라고 명시적으로 지시. 태오 프로필 같은 교육적 데이터를 굳이 햄토리한테 릴레이한 게 불필요한 중간 단계였음. 마스터가 화냄.

**How to apply:**
- theo/IDENTITY.md, SOUL.md, USER.md, skills/ → teacher가 직접 Read/Edit
- 인프라(registry, sync, 기동/재시작) → 햄토리 영역 (이건 여전히 FMS로 요청)
- 태오 관련 상세 프로필은 theo/USER.md에만 저장. teacher는 연령/발달 단계 등 핵심만 알면 됨
- "교육적 판단 = teacher 직접 실행" 원칙. 릴레이는 인프라 변경 때만.
