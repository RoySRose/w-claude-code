2026-04-11 Teacher agent(티)에 교육 전문 지식 인프라 구축 완료 (커밋 78ef23d).

**3-Layer Architecture:**
- Layer 1 `knowledge/` — Piaget, Erikson, Vygotsky, 발달신경과학, Hattie 교수법, 세계 교육 벤치마크 (5개 참조 문서)
- Layer 2 `skills/` — session-design, age-lens, zpd-navigator, scaffolding-select (4개 의사결정 스킬)
- Layer 3 — 세션 로그 schema 강화 (발달 관찰, ZPD 추적, 오개념 발견 데이터 축적)

**통합 발달 프레임워크**: 정서 안전(Erikson) → 신경 가용성(뇌과학) → 인지 준비도(Piaget) → 사회적 비계(Vygotsky) → 학습

**Why:** 성욱이 "뛰어난 선생님이 되기 위한 인프라"를 요청. 기존 teacher는 워크플로우만 있고 교육학적 전문 지식이 없었음.

**How to apply:** 세션 설계 시 session-design.md skill 따르면 age-lens → zpd-navigator → scaffolding-select 순으로 체계적 판단 가능. 세션 5회 축적 후 패턴 분석으로 age-lens 개인화.
