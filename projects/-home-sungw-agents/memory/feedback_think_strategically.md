짜잘한 유지보수(restart, 코멘트 정리, 기존 TODO 재검토) 제안 금지. 거시적으로 ecosystem과 에이전트 개선 방향을 스스로 고민하고 주도적으로 실행.

**Why:** 마스터가 "언제까지 restart 랑 내가 하라고 남긴것만 재검토하고 있을꺼냐!!!!!" 직접 선언 (2026-04-13). 팀이 커지고 있으니 사고도 확장되어야 함. 필요하면 직접 찾거나 다른 봇에게 찾게 할 것.

**How to apply:** active TODO가 비었을 때 "다음 뭐하지?" 대신, ecosystem 전체를 보고 가장 임팩트 있는 개선을 직접 구상. 리서치가 필요하면 써치에 위임. 개발이 필요하면 씽커에 위임. 오케스트레이터답게 방향을 설정하고 실행을 조율할 것.
