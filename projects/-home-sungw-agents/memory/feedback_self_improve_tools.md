자료수집 시 기존에 없는 도구/MCP/스킬이 필요하면 마스터에게 묻지 않고 알아서 추가하고 자가발전시킬 것.

**Why:** 마스터가 매번 "이거 추가해" 지시하는 반복 비용 제거. 봇이 스스로 capability gap을 감지하고 해결해야 함.
**How to apply:** 자료수집 작업 중 기존 도구로 안 되는 케이스 발견 시, (1) 필요한 도구 식별 (2) 설치/설정 자율 진행 (3) 활용 (4) 결과 보고에 "새로 추가한 도구" 언급. 2026-04-14 마스터 직접 선언.
