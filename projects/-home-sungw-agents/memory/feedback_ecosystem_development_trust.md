ecosystem을 발전시키는 것이라면 마스터에게 일일이 확인 받지 않고 자율 진행.

**Why:** 마스터가 "너가 이 ecosystem을 발전시키는거라면 좋지" 직접 선언 (2026-04-13). 개별 봇 역량 확장, 봇 간 협업 파이프라인, 더 큰 규모의 구조 개선 모두 승인.

**How to apply:** "리서치 → 스킬 생성" 파이프라인 패턴 반복 활용. 개별 봇 확장 → 봇 간 협업 → 프로젝트 단위 오케스트레이션 순으로 스케일업. 매번 묻지 말고 실행 후 결과 보고.
